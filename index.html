MORSE OPERATOR

Signal Corps · 1939–1945

Master Morse code through a WWII-themed training program. Graduate Boot Camp, complete missions, and climb the ranks from Recruit to General.

Boot Camp

14 progressive lessons covering all 26 letters and 10 numbers. Each lesson includes drills, field exercises, and a Morse Match minigame. Pass the Final Exam to unlock missions.

4 Mission Types

  • Radio Operator TRANSMIT
    Transmit written messages letter-by-letter in Morse code across three difficulty tiers.
  • Code Breaker INTERCEPT
    Crack a 5-letter word transmitted in Morse with 5 attempts.
  • Explosive Ordnance BOMB DISPOSAL
    Defuse multi-stage bombs by decoding Morse clues under time pressure.
  • Signal Overflow SURVIVE
    Real-time endurance mode where outgoing messages queue faster and faster. Earn power-ups to survive.

Rank Progression

Earn XP from missions and advance through military ranks. Track your accuracy, speed, and performance across all game modes in your Service Record.
